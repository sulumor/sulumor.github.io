POKE MAGASIN

Site E-commerce factice sur les pokémons de la 1ère génération (les 151 premiers).

À propos

A savoir sur ce projet

Ce site m'a permis de pratiquer VueJS 2 en Option API et les mécaniques d'un site E-commerce. Ce fut aussi pour moi la découverte de l'utilisation des frameworks. Utilisation d'Axios afin de faire l'appel à l'API pokeapi. Mise en place du store avec VueX mais aussi d'une page d'attente pendant la récupération des données. Il reprend aussi les principales mécaniques de ce type de site c'est à dire la récupération de donnée via une API, mise en place de fiche pour chaque 'produit', possibilité de filtrer, mise dans un panier, gestion des stocks.

Les compétences déployées

VueJS 2 Option API VueX Vue Router Axios Filtre via une sélection ou bien une recherche
