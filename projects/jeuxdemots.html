JEUX DE MOTS

Site regroupant 2 jeux, le Pendu et l'Anagramme avec 2 niveaux de difficulté : enfant, sur une base de mots propre au site, et adulte, trouvant des mots aléatoires via une API.

A savoir sur ce projet

Le projet est développé avec VueJS 3. Ne trouvant pas d'API me générant de mots pour une enfant de 6/7 ans, je prend le partie de créer 2 niveaux, enfant et adulte. Le niveau enfant chosit un mot dans un fichier JSON que j'ai créé. Le niveau adulte va, quant à lui, faire appel à l'API trouve-mot . En gardant la même base de mot, j'ai ajouter un jeu d'anagramme avec un temps pour trouver le bon mot.
Ce projet m'a permis de découvrir Pinia, PrimeVue et PrimeFlex mais aussi d'appronfondir mes connaissances sur VueJS, l'utilisation du Router, la gestion du temps de chargement pour l'expérience utilisateur mais aussi les stores.
À la fin de chaque partie, le joueur a la posibilité de sauvegarder son score. Cela se fait sur une base de données via Supabase.

Les compétences déployées

VueJS 3 Composition API Pinia Vue Router Fetch API TypeScript BDD Filtre d'un tableau Utilisation librairie de composants
